General:

Activate on all websites

Toggles the add-on automatically on every website possible.
When this is disabled, the app will enable only on whitelisted websites, or when pressing the "C" icon in the add-on bar.

Smooth transitions

Makes the text color and font size change gradually, rather than instantly. May not work on all sites.

Global options:

Strength

Adjusts the detection of dark backgrounds. Lower this value when the app dims light text against dark backgrounds.

Increase font (%)

Increases font size. Fonts above the size threshold won't be affected.

Size threshold (px)

Determines the minimum size a font has to be (in pixels) for it to be affected by an increase in size. By default, only small fonts are affected.

Avoid colorful text

Attempts to avoid colorful text and links, if they are considered readable.

Links have a slightly higher threshold, i.e. the same color might not be dimmed if it's a link.
Can be adjusted with strength.

Bold text

Changes the font weight of all text to bold.

Skip headings

Avoids dimming all big headings. (<h1>, <h2>, <h3>)

Skip white text

Avoids dimming all white text.
White text is usually behind images, where determining the background color is not always possible.

Underline black links

Adds an underline to links that turn black with the add-on enabled, to distinguish them from other black text.

Advanced mode

Dims text to a darker shade of its color, instead of just black.
e.g. Light grey turns dark grey.
The effect can be adjusted with the strength slider.

This option may cause slowdowns on big websites.

Force max opacity

Some websites use dark fonts that still look grey because of low opacity. This option forces full opacity on all text.

Disable this on websites that use opacity to hide some elements (e.g. YouTube).

Force black placeholder

Forces all placeholder text to turn black, instead of just increasing their opacity. Useful in websites where placeholders are still grey with the add-on enabled.
