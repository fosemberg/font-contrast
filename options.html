Global options:

Strength

Higher strength may detect more grey text, but may also dim light fonts against dark backgrounds.
This value is overridden by the Strength of a whitelisted website.

When advanced dimming is enabled, this value also controls font dimming amount.

Increase size (%)

Increases font size. Fonts above the size threshold won't be affected.

Size threshold (px)

Determines the minimum size a font has to be (in pixels) for it to be affected by a size increase. By default, only small fonts are affected.

Activate on all websites

Toggles the add-on automatically on every website possible.
When this is disabled, the app will enable only on whitelisted websites, or when pressing the "C" icon in the add-on bar.

Advanced dimming

Dims text to a darker shade of its color.
e.g. Light grey turns dark grey, instead of black.
The effect can be adjusted with the strength slider.

This feature is experimental and may cause slowdowns on some websites.

Avoid readable text

Attempts to avoid readable text and links.
Affected by strength.

Skip headings

Avoids dimming all big headings. (<h1>, <h2>, <h3>)

Bold text

Changes the font weight of all text to bold.

Force max opacity

Some websites use dark fonts that still look grey because of low opacity. This option forces full opacity on all text.

Disable this on websites that use opacity to hide some elements (e.g. YouTube).

Force placeholder dimming

Forces all placeholder text to turn black, instead of just increasing their opacity. Useful in websites where placeholders are still grey with the add-on enabled.

Smooth transitions

Makes the text color and font size change gradually, rather than instantly. May not work on all sites.
